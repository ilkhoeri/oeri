| API           | Type                                              | default                    | Annotation                             |
| ------------- | ------------------------------------------------- | -------------------------- | -------------------------------------- |
| el`?`         | React.ElementType                                 | `wrap:"div"` `inner:"div"` | `el={{ wrap: "div", inner: "div" }}`   |
| direction`?`  | (string & {})                                     | `"right-to-left"`          | set running direction types            |
